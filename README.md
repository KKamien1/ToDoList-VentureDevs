# ToDo List

This project was bootstrapped with [Create React App](https://github.com/facebookincubator/create-react-app).

## Start app with

```
npm start
```

or

```
yarn start
```

## Keyboard Shortcuts

- Enter - add new item (if not empty)
- a : shows all todos
- d : shows 'done'
- u : shows 'undone'
- t : toggle
